# badges

Code badges for code repositories

## Badge Names Explained

`<Repository Name>-<identifier>`

| Idenfitier   | Meaning                             |
| ------------ | ----------------------------------- |
| cc           | code coverage                       |
| ut           | unit testing results                |
| wf           | work flow build job                 |
| re           | Release                             |
| dependencies | scan dependencies last verified run |
